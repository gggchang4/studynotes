# 参考课程映射

## 本笔记的学习目标
- 记录三个参考课程仓库的结构特点，并说明它们如何被吸收到本知识库中。

## 参考仓库
- Hugging Face Agents Course：https://github.com/huggingface/agents-course.git，本地参考 commit：de076ef
- Datawhale Hello Agents：https://github.com/datawhalechina/hello-agents.git，本地参考 commit：248aa24
- Microsoft AI Agents for Beginners：https://github.com/microsoft/ai-agents-for-beginners.git，本地参考 commit：764581f

## 核心内容
- Hugging Face 课程强调从 LLM 到 Agent 的动手路径，重点包括 Thought-Action-Observation 循环、ReAct、Tools、smolagents、LlamaIndex、LangGraph、Agentic RAG、GAIA、函数调用微调、评估与可观测性。
- Datawhale 课程更适合作为中文主教材，覆盖智能体发展史、LLM 基础、经典 Agent 范式、低代码平台、框架开发、自建 Agent 框架、Memory/Retrieval、上下文工程、通信协议、Agentic RL、性能评估和多个综合项目。
- Microsoft 课程偏工程设计模式，重点包括 Tool Use、Planning、Multi-Agent、Metacognition、Agentic RAG、Trustworthy Agents、Production、MCP/A2A/NLWeb、Context Engineering、Memory、Browser Use、Securing Agents。
- 本知识库没有按仓库逐章搬运，而是把三套课程共同覆盖的内容归入核心笔记，把课程没有系统展开但工程必需的内容放入工程化目录。
