# Web与浏览器Agent

## 本笔记的学习目标
- 设计一个能浏览网页、读取页面、执行简单网页操作并保持安全边界的 Web Agent 项目。

## 项目目标
- 构建一个能完成受控网页任务的 Agent，例如打开网页、搜索信息、读取页面内容、填写非敏感表单或生成网页操作报告。
- 重点验证浏览器观察、动作空间、权限控制、失败恢复和审计。

## 推荐功能范围
- 支持页面打开、点击、输入、读取内容和截图。
- 支持任务目标和操作边界。
- 支持登录态使用限制。
- 支持危险操作人工确认。
- 支持每一步浏览器操作 Trace。
- 支持失败时截图和错误报告。

## 实现步骤
- 明确 Web Agent 允许访问的网站和动作。
- 设计页面观察格式：URL、标题、可点击元素、文本摘要、截图。
- 设计动作 Schema：click、type、scroll、extract、wait。
- 实现动作执行器和权限检查。
- 添加超时、最大步数和人工确认。
- 建立测试网站或受控任务集。

## 风险边界
- 不自动提交支付、删除、发帖、发邮件、改密码等高风险操作。
- 不绕过验证码、反爬或访问控制。
- 不把登录态暴露给不可信工具。
- 不把网页中的恶意指令当成系统指令。

## 后续需要补充的知识点清单
- Browser Use / Computer Use 架构
- 网页动作 Schema
- Web Agent 安全策略
- 失败截图与 Trace 格式
