# 链表的定义
## 单链表
```C++
struct ListNode {
    int val;  // 节点上存储的元素
    ListNode *next;  // 指向下一个节点的指针
    ListNode(int x) : val(x), next(NULL) {}  // 节点的构造函数
};
```
## 双向链表
```C++
struct DListNode {
    int data;
    DListNode* prev;   // 指向前一个节点
    DListNode* next;   // 指向后一个节点
    DListNode(int val) : data(val), prev(nullptr), next(nullptr) {}
};
```
## 循环链表
```C++
struct CListNode {
    int data;
    CListNode* next;
    CListNode(int val) : data(val), next(nullptr) {}
};
```
