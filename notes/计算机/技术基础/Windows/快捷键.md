# Windows 常用快捷键总结（Windows 10 / 11）

## 一、窗口与任务切换（高频）

| 快捷键 | 功能 |
|------|------|
| Alt + Tab | 在已打开的应用之间切换 |
| Alt + Shift + Tab | 反向切换应用 |
| Win + Tab | 打开任务视图（窗口 + 虚拟桌面） |
| Win + T | 依次选中任务栏程序 |
| Alt + F4 | 关闭当前窗口 |

---

## 二、虚拟桌面管理（Windows 10 / 11）

| 快捷键 | 功能 |
|------|------|
| Win + Ctrl + D | 新建虚拟桌面 |
| Win + Ctrl + ← / → | 切换虚拟桌面 |
| Win + Ctrl + F4 | 关闭当前虚拟桌面 |

---

## 三、窗口布局与分屏（办公必备）

| 快捷键 | 功能 |
|------|------|
| Win + ← / → | 窗口左右分屏 |
| Win + ↑ / ↓ | 最大化 / 最小化窗口 |
| Win + Shift + ← / → | 在多显示器间移动窗口 |
| Win + Home | 最小化除当前窗口外的所有窗口 |

---

## 四、桌面与系统操作

| 快捷键 | 功能 |
|------|------|
| Win + D | 显示 / 恢复桌面 |
| Win + E | 打开文件资源管理器 |
| Win + I | 打开系统设置 |
| Win + L | 锁定电脑 |
| Win + R | 打开“运行”对话框 |
| Win + X | 打开高级系统菜单 |

---

## 五、搜索、命令与系统管理

| 快捷键 | 功能 |
|------|------|
| Win | 打开开始菜单 |
| Win + S | 系统搜索 |
| Ctrl + Shift + Esc | 直接打开任务管理器 |
| Ctrl + Alt + Del | 系统安全界面 |

---

## 六、文本编辑与通用操作（通用规则）

| 快捷键 | 功能 |
|------|------|
| Ctrl + C | 复制 |
| Ctrl + V | 粘贴 |
| Ctrl + X | 剪切 |
| Ctrl + Z | 撤销 |
| Ctrl + Y | 重做 |
| Ctrl + A | 全选 |
| Ctrl + S | 保存 |
| Ctrl + F | 查找 |

---

## 七、浏览器 / 程序内常用快捷键

| 快捷键 | 功能 |
|------|------|
| Ctrl + Tab | 切换标签页 |
| Ctrl + Shift + Tab | 反向切换标签 |
| Ctrl + W | 关闭当前标签 |
| Ctrl + T | 新建标签 |
| Ctrl + N | 新建窗口 |

---

## 八、截图与录屏（非常实用）

| 快捷键 | 功能 |
|------|------|
| Win + Shift + S | 截图工具（区域 / 窗口） |
| Win + PrtSc | 全屏截图并自动保存 |
| PrtSc | 全屏复制到剪贴板 |
| Alt + PrtSc | 当前窗口截图 |
| Win + G | Xbox Game Bar（录屏） |

---

## 九、输入法与显示相关

| 快捷键 | 功能 |
|------|------|
| Ctrl + Shift | 切换输入法 |
| Win + Space | 切换输入法 / 语言 |
| Win + + / - | 放大镜 |
| Win + P | 投影 / 显示模式切换 |

---

## 十、速查高频组合（建议优先记住）

- Alt + Tab —— 窗口快速切换  
- Win + ← / → —— 分屏办公  
- Win + E —— 文件资源管理器  
- Win + R —— 运行命令  
- Ctrl + Shift + Esc —— 任务管理器  
- Win + Shift + S —— 快速截图
