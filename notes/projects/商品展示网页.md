# 开发技术栈

## 后端
Spring Boot + Spring Web + MyBatis / JPA + MySQL + Lombok

JAVA框架：Spring Boot
网络接口：Spring Web
ORM框架：JPA
数据库：MySQL
工具：Lombok

## 前端
Vue3 + Pinia+ Vue Router + Axios + Element Plus

前端框架：Vue3 (MVVM架构)
全局状态管理工具：Pinia
路由库：Vue Router
HTTP请求库：Axios
组件库：Element Plus
